The Big Announcement

About

The Big Announcement is an uncensorable website that delivers a single message, authored by the individual with the deepest pockets (the most ETH).

Developed during HackFS 2020, it is a minimal project that demonstrates the integration of Ethereum and IPFS with the web.

The source code is on GitHub.
